План разработки Coin Flip Betting - Отчет о прогрессе
Версия 1.3 | Дата: 14 декабря 2023 | 15:00

1. Разработка минимального смарт-контракта для MVP
   - Создание простого контракта для ставок на Solidity.
   - Реализация функций для размещения ставок и разрешения ставок.
   - Локальное тестирование контракта.
   - Деплой на тестовую сеть Moonbase Alpha.
   - **Хранение истории ставок**:
     - Создание структуры данных для хранения информации о каждой ставке, включая адрес игрока, сумму ставки, выбор (орёл или решка), время ставки, статус (разрешена или нет) и результат подбрасывания.
     - Использование массива для хранения всех ставок, что позволит легко добавлять новые ставки и отслеживать их.
     - Реализация функции для размещения ставки, которая будет добавлять новую запись в массив с информацией о ставке.
     - Реализация функции для разрешения ставки, которая будет обновлять статус и результат ставки после подбрасывания монеты.

2. Интеграция интерфейса ставок
   - Завершение и интеграция компонента для ставок (BetForm).
   - Добавление возможности выбора между орлом и решкой.
   - Добавление поля для ввода суммы ставки.
   - Добавление кнопки для подтверждения ставки.

3. Проверка и интеграция с расширением Chrome
   - Реализация проверки наличия установленного расширения.
   - Интеграция компонента для уведомлений о расширении.
   - Разработка интерфейса для получения результатов от расширения.
   - Добавление обработчиков событий для взаимодействия с расширением.

4. Базовая интеграция с контрактом
   - Настройка ABI и адреса смарт-контракта для взаимодействия с фронтендом.
   - Реализация функции для размещения ставки, которая будет отправлять данные в смарт-контракт.
   - Обработка результата подбрасывания монеты и отправка его в смарт-контракт.
   - Отображение результата ставки и обновление баланса игрока.
   - **Доступ к истории ставок**:
     - Реализация функции для получения информации о всех ставках или конкретной ставке по идентификатору, что позволит пользователям отслеживать свои ставки и результаты.

План разработки Coin Flip Betting - Отчет о прогрессе
Версия 1.4 | Дата: 14 декабря 2023 | 15:30

1️⃣ Разработка минимального смарт-контракта для MVP
Создание простого контракта для ставок на Solidity.
Реализация функций:
Размещение ставки (принимает сумму и выбор игрока).
Разрешение ставки (обновляет результат и отправляет выигрыш).
Локальное тестирование контракта.
Деплой на тестовую сеть Moonbase Alpha.
Хранение последней ставки:
Контракт хранит только одну ставку (адрес игрока, сумму, выбор, результат).
Как только игрок делает новую ставку, данные затираются.
✅ Плюсы:
Низкие затраты на газ (история не накапливается).
Быстрый доступ к последней ставке.
2️⃣ Интеграция интерфейса ставок
Завершение и интеграция BetForm.
Выбор между орлом и решкой.
Поле для ввода суммы ставки.
Кнопка "Подтвердить ставку".
3️⃣ Проверка и интеграция с расширением Chrome
Проверка наличия установленного Coin Flip.
Интеграция уведомлений о статусе расширения.
Получение результата подбрасывания.
Обработчики событий для работы с расширением.
4️⃣ Базовая интеграция с контрактом
Настройка ABI и адреса контракта.
Реализация функции размещения ставки (отправка данных в смарт-контракт).
Обработка результата подбрасывания и отправка в контракт.
Обновление баланса игрока и отображение результата.
